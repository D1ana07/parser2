# VK Ads Cases Parser

## Описание
Парсер для сбора бизнес-кейсов с сайта VK Ads (adz.vk.com/cases).

## Функциональность
- Извлекает названия кейсов
- Получает ссылки на полные кейсы
- Парсит даты публикации
- Нормализует даты к формату YYYY-MM-DD
- Сохраняет в JSON и CSV форматах
